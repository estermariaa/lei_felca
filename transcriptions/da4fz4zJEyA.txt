 A linha K com 610 para restriados e aquecimento e 630 para linha de congelados. Primeiramente para ajustar o 7-Pont você dará um clique no botão 7, assim ajustando para cima e para baixo, 7 para salvar e depois o 7-Pont econômico, o mesmo jeito. Para cima e para baixo mudar o valor e 7 para salvar. Para forçar um desgelo você vai segurar o botão do desgelo até ativá-lo. Nesse momento o LED de desgelo irá acender e do mesmo jeito para desligar remotamente o desgelo. Assim você forçou o final do desgelo. Para acessar as configurações você vai segurar o botão de 7, aparecerá P01 que é a senha de acesso, você vai botar ela que padrão da senha de acesso e dá um 7, assim você liberou para modificar qualquer configuração. 7 você acessa a configuração, mudando o valor e 7 para salvar. Automaticamente ele voltará para a tela de temperatura inicial ou no botão do 7 você retorna para o início. Para acessar algumas informações do controlador como temperatura, máxima e mínima e tempo de processo você irá usar teclas de atalho. Para visualizar a temperatura máxima você irá dar um toque na tecla para cima, mostrando máxima e mínima de temperatura de sensor 1, máxima e mínima de temperatura de sensor 2. Caso esteja conectado mostrará também máxima e mínima de temperatura de sensor 3. Para visualizar o tempo de processo que está acontecendo você dará um clique para baixo e mostrará qual processo, refugeração, 2 minutos trabalhando e logo após também mostrará a temperatura real do sensor 2, caso conectado e do sensor 3 caso conectado.
